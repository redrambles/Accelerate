== Description ==

Displays the current template file (and template parts if any) in WordPress admin toolbar

= Translations included =
* German
* Greek
* Portuguese (Brazil)

== Installation ==

1. Upload the entire current-template-file folder to the /wp-content/plugins/ directory.
2. Activate the plugin through the ‘Plugins’ menu in WordPress.

== Frequently Asked Questions ==

= Where can i see the current template file? =

It appears in the admin toolbar, if it is enabled.
